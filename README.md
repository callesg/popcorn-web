# popcorn-web
A web-interface for the popcorn time api, connect with ffmpeg and rtorrrent for a seamless experience.

## Install
In your webfolder belonging to your php webserver 
```bash
git clone https://github.com/callesg/popcorn-web
cd popcorn-web
mkdir db
chmod 777 db
```
